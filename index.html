HTML

  • Enter your HTML notes here
  • DOCTYPE html
  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • title element defines the title for a webpage. This element is required and is extremely important for Search Engine Optimization (SEO).
  • style element defines the CSS styles associated with a website.
  • base element defines the base URL for a webpage.
  • link element connects an external resource to the HTML document.
  • meta element defines metadata such as the character set, description, keywords, author, and viewport.
  • script element embeds client-side scripts in the HTML document.
  • The h1, h2, h3, h4, h5, and h6 elements represent the level of heading a given text block represents.
  • p element represents a paragraph or block of text. You'll use this element extensively to wrap most of the text on your webpages.
  • The ul, ol, and li elements represent unordered lists, ordered lists, and list items. In essence, these HTML elements represent bulleted or numbered lists.
  • img element contains information about images that are displayed on the webpage.
  • br element creates a line of empty space, or a line break, between two blocks of content.
  • section elements are exactly what we need to get the job done here! They will allow us to elegantly organize the content on our page based on the topics we have.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.

Git

    • git status: checks what branch we are currently on
    • git checkout -b branch-name: creates a new branch and switches to it
    • git checkout just checks out an active branch
    • git add -A stages changes
    • git commit -m "added code to HTML file" commits the changes with -m being the message
    • git pull origin main pulls the origin in so you can insert the change
    • git push origin feature/add-html pushes the change that was in the origin

JavaScript

    • A variable is a named container that allows us to store data in our code.
    • Control flow is the order in which a computer executes code in a script.
